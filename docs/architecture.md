# Архитектура расширения Python Snippet Generator для VS Code

## Краткое описание архитектуры

Расширение построено на стандартной модели VS Code:

- **Активатор**: модуль активации (`extension.ts`) регистрирует команду для генерации сниппетов.
- **Менеджер команд**: после вызова команды пользователем открывается диалог выбора типа сниппета (Quick Pick).
- **Генератор сниппетов**: на основе выбранного элемента и введённых параметров формируется текст кода с вычислением позиции курсора.
- **Редактор**: вставка сниппета происходит в активное окно редактора, а курсор смещается в ключевую позицию.

## Основные компоненты

- **extension.ts**  
  - Инициализация и регистрация команды
  - UI выбора типа сниппета
  - Генерация текста и обработка ввода параметров
  - Вставка текста и корректировка позиции курсора
  - Уведомления и обработка ошибок

- **package.json**  
  - Метаинформация о расширении
  - Команды, события активации, описание

- **docs/**  
  - Хранение документации по архитектуре, использованию, тестам

- **src/**  
  - Реализация всех рабочих функций расширения
