# Документация расширения Python Snippet Generator для VS Code

## Описание

**Python Snippet Generator** — расширение для Visual Studio Code, предназначенное для быстрого добавления популярных шаблонов (сниппетов) кода на Python в открытый редактор. Плагин поддерживает интерактивный выбор сниппета, запрос параметров и установку курсора для мгновенного перехода к редактированию.

---

## Основные функции и возможности

- Мгновенное добавление типовых сниппетов Python: `if`, `for`, `while`, функции, классы и другие.
- Удобный выбор типа шаблона через выпадающее меню (Quick Pick).
- Запрос дополнительных параметров в интерактивных окнах (имя переменной цикла, условие, параметры функции и т.д.).
- Корректная установка курсора после вставки сниппета.
- Сообщения об успехе или ошибке при вставке.
- Легко расширяемый список шаблонов.

## Основная архитектура проекта

**Файловая структура:**

```
python-snippet-generator/
│
├── .vscode/               # Конфигурация среды разработки
├── docs/                  # Документация (documentation.md, architecture.md)
├── node_modules/          # Зависимости проекта
├── src/                   # Исходный код плагина (extension.ts)
├── package.json           # Манифест расширения, команда, описание, точки входа
├── README.md              # Краткое описание проекта и инструкции по установке
├── test/                  # (опционально) тесты
└── ...                    # Прочие служебные файлы (tsconfig.json, gitignore и т.д.)
```

**Логика работы:**

- **extension.ts** — запускает активацию расширения, регистрацию команды, основной UI для выбора и вставки сниппетов.
- **package.json** — определяет команды (`insertSnippet`), условия активации и описание для VS Code Marketplace.
- **docs/documentation.md** — подробная пользовательская и техническая документация для расширения.

## Поддерживаемые сниппеты и описание

- `if` — Базовая конструкция `if`
- `if-else` — Блок `if-else`
- `if-elif-else` — Блок `if-elif-else`
- `for` — Цикл for (запрашиваются имя переменной и выражение для итерирования)
- `while` — Цикл while (запрашивается условие цикла)
- `function` — Определение функции (имя и параметры)
- `class` — Определение класса (имя и наследование)
- `try-except` — Блок обработки исключений
- `with` — Контекстный менеджер с переменной
- `main` — Блок для запуска модуля через `if __name__ == "__main__"`

---

## Обычный сценарий работы

1. Откройте файл Python в редакторе VS Code.
2. Вызовите команду “Python Snippet Generator: insertSnippet” через командную палитру (Ctrl+Shift+P).
3. Выберите нужный шаблон (сниппет) в окне выбора.
4. Введите дополнительные параметры при необходимости (например, имя переменной или условие).
5. Сниппет автоматически вставится в место текущего курсора; курсор будет установлен после ключевых строк.
6. Продолжайте редактирование кода прямо с нужного места.

## Примеры вставляемых шаблонов

- **if:**
  ```python
  if condition:
      pass
  ```

- **if-else:**
  ```python
  if condition:
      pass
  else:
      pass
  ```

- **if-elif-else:**
  ```python
  if condition1:
      pass
  elif condition2:
      pass
  else:
      pass
  ```

- **for:**
  ```python
  for i in range(10):
      pass
  ```
  *(Имя переменной и итерируемый объект — запрашиваются при вставке)*

- **while:**
  ```python
  while True:
      pass
  ```
  *(Условие цикла — запрашивается при вставке)*

- **function:**
  ```python
  def my_function(param1, param2):
      """
      Описание функции
      """
      pass
  ```
  *(Имя функции и параметры — запрашиваются при вставке)*

- **class:**
  ```python
  class MyClass(ParentClass):
      """
      Описание класса
      """

      def __init__(self):
          pass
  ```
  *(Имя класса и родительский класс — запрашиваются при вставке)*

- **try-except:**
  ```python
  try:
      pass
  except Exception as e:
      print(f"Произошла ошибка: {e}")
  ```

- **with:**
  ```python
  with open("file.txt") as file:
      pass
  ```
  *(Контекстный менеджер и имя переменной — запрашиваются при вставке)*

- **main:**
  ```python
  if __name__ == "__main__":
      pass
  ```
